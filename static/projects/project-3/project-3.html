Project Description

Situation:

The project focuses on detecting and classifying harmful brain activity, specifically seizures, using Convolutional Neural Networks (CNNs). Epilepsy is a neurological disorder characterized by recurring seizures caused by unusual brain activity, which is challenging to detect and diagnose accurately. The project aims to automate the detection process and classify the types of seizures using EEG signals.

Task:

The primary task was to develop a system capable of automatically detecting and classifying different types of seizures from EEG data. The goal was to use CNN algorithms to convert EEG signals into images and identify the seizure types, improving the early detection and treatment of epilepsy.

Action:

Dataset Selection and Preparation: The team used the TUH EEG Dataset due to its extensive collection of EEG recordings categorized into different seizure types. They preprocessed the data by converting EEG signals from the European Data Format (EDF) to spectrograms using the MNE-Python library. They also cleaned and augmented the data to ensure accuracy.

Model Development: Various CNN architectures were implemented, including ResNet50V2, ResNet152V2, ResNet101, EfficientNetB0, and DenseNet201. Both pre-trained and non-pretrained models were tested. The models were trained using the Adam optimizer with varying learning rates.

Model Evaluation: The models were evaluated based on their training and validation accuracy. The DenseNet201 model showed better outcomes, achieving a high training accuracy but indicating slight overfitting.

Result:

The project demonstrated that CNNs could be used to classify different types of seizures using EEG data converted into spectrograms. The DenseNet201 model achieved the highest training accuracy (87.81%) but showed some overfitting with a validation accuracy of 59.38%. The EfficientNetB0 model also performed well, particularly when using transfer learning.

Skills and Technologies Implemented:

  • Data Science and Machine Learning: Knowledge of CNNs, data preprocessing, and model evaluation.
  • Python Programming: Used for data processing, model training, and evaluation.
  • Libraries and Tools: TensorFlow, MNE-Python, and various deep learning frameworks for model implementation.
  • Data Handling: Experience in handling large datasets, data augmentation, and cleaning.

Reflection:

The project successfully created a baseline for classifying EEG signals using CNNs, highlighting the potential of deep learning in medical diagnostics. However, there is a need for further improvements to enhance validation accuracy and reduce overfitting. Future work should focus on creating a more comprehensive dataset, utilizing advanced deep learning models, and potentially developing an application for real-time seizure detection. This project provided valuable insights into the complexities of EEG data and the effectiveness of different CNN architectures in medical applications.
